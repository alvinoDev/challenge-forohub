# challenge-forohub
API REST para una plataforma de foro de discusión desarrollada con Java 25 y Spring Boot 4.0.1. El sistema implementa un CRUD completo para la gestión de tópicos, persistencia en MySQL, migraciones de base de datos con Flyway y un robusto esquema de seguridad mediante Spring Security y Tokens JWT para la autenticación y autorización de usuarios.
